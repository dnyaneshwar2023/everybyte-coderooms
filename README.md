# everybyte-coderooms

Everybyte Coderooms makes your coding discussions easy using collaborative code editor and features like running the code, Voice Chat, Saving the code for further use and many more...

Technologies Used:

- React.js
- Node.js
- Socket.io
- Nodemailer
- OAuth 2.0
- Redis Pub/Sub
